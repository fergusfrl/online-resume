NZ River Flows

NZ River Flows

My first website! I jumped in the deep end with learning html, css and javascript. The site retrieves flow data from regional council databases and displays them on a map. Markers are colour coded depending on whether they are able to be paddled or not. This was an exciting project which I will be continuing with in the future.

Click here to take a look!



PreLoad

Preload

Preload is my group project for COSC345 - Software Engineering, at the University of Otago. Preload is an iOS app that finds and calculates the best 'dollar to alcohol' percentage in your area. The code is written in C and wrapped in swift to function on iOS devices. It's a fun project and I've learned heaps!

Click here to take a look at the GitHub page!

BI Dashboards

BI Dashboards

My project at TracMap (see past experience), was to develop a BI dashboard solution for clients to use. The software I used was SpagoBI, an open source BI platform.
